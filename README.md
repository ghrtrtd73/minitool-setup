Physical Disk ‚Üí Master Boot Record ‚Üí Primary Partitions ‚Üí Logical Drives

3. **Boot Sector**: Contains code to load operating system

### What's the difference between MBR and GPT?

#### **Partition Table Comparison**:
| Feature | **MBR (Master Boot Record)** | **GPT (GUID Partition Table)** |
|---------|------------------------------|--------------------------------|
| **Maximum Disk Size** | 2TB | 18 Exabytes |
| **Primary Partitions** | 4 partitions | 128 partitions |
| **Boot Support** | BIOS only | UEFI (and BIOS with compatibility) |
| **Data Protection** | Single boot record | Multiple copies, CRC protection |
| **Operating System** | All Windows versions | Windows 8/10/11, Linux, macOS |

#### **When to Use Each**:
```yaml
Choose MBR when:
  - Using older hardware (pre-2010)
  - Need compatibility with older OS
  - Disk size under 2TB
  - Legacy BIOS systems

Choose GPT when:
  - Modern hardware (UEFI systems)
  - Disks larger than 2TB
  - Need more than 4 partitions
  - Want better data protection
  - Using Windows 10/11
```
